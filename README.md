# Expense Tracker

A Python application for managing daily expenses using a simple menu-driven interface. The project allows users to add, view, delete, and summarize expenses while storing records in CSV files.

## Features

- Add expense entries
- View all expenses
- Delete saved expenses
- Category-wise summaries
- Input validation
- CSV file storage

## Project Structure

The application is divided into separate modules for menu handling, validation, file operations, expense management, and reporting. This structure improves readability and keeps responsibilities separated across files.
